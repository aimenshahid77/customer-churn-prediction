# Customer Churn Prediction

## Overview
A machine learning model that predicts whether a customer will leave a subscription service based on behavioral signals.

## Dataset
Customer Churn Dataset from Kaggle — 440,833 customer records.

## Tech Stack
- Python
- Google Colab
- Pandas
- Scikit-learn
- Matplotlib & Seaborn

## ML Approach
- Algorithm: Random Forest Classifier (100 trees)
- Accuracy: 99.99%
- Train/Test Split: 80/20

## Real World Usecase
Telecom companies, streaming platforms, and SaaS businesses can use this model to identify at-risk customers before they cancel, enabling proactive retention strategies.
